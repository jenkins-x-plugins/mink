### Linux

```shell
curl -L https://github.com/jenkins-x-plugins/mink/releases/download/v{{.Version}}/jx-mink-linux-amd64.tar.gz | tar xzv 
sudo mv jx-mink /usr/local/bin
```

### macOS

```shell
curl -L  https://github.com/jenkins-x-plugins/mink/releases/download/v{{.Version}}/jx-mink-darwin-amd64.tar.gz | tar xzv
sudo mv jx-mink /usr/local/bin
```
